## Current shortcuts

- `Ctrl/Cmd + K` -> trade console
- `Ctrl/Cmd + G` -> admin panel

## Intent

These shortcuts are navigational only right now.

That is deliberate. The client does not yet expose destructive trading actions directly on keyboard shortcuts.

## Still to define

- in-trade keybind map
- explicit confirmation rules for destructive actions
- discoverability overlay
- accessibility and focus-handling constraints
