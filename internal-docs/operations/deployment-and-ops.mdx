## Deployed shape

- one ECS/Fargate service for the Next.js client
- one ALB in front of both client and backend paths
- one EC2 instance for the Rust exchange core and market-data services
- one local PostgreSQL instance on the EC2 host
- one systemd service named `exchange`
- one systemd service named `market-data-service`
- GitHub-backed checkout at `/home/ec2-user/exchange-v2`

## Important paths

- repo: `/home/ec2-user/exchange-v2`
- backend env: `/home/ec2-user/exchange-v2/exchange.env`
- binary: `/home/ec2-user/exchange-v2/exchange/target/release/exchange`
- market-data binary: `/home/ec2-user/exchange-v2/exchange/target/release/market-data-service`
- ECS stack: `exchange-client`
- ALB hostname: `exchange-client-alb-1466111370.us-east-2.elb.amazonaws.com`

## Update procedure

From your local machine:

```bash
git push origin main
AWS_PROFILE=personal infra/client-ecs/build_and_push.sh
IMAGE_URI=490004617163.dkr.ecr.us-east-2.amazonaws.com/exchange-client:TAG \
infra/client-ecs/deploy_stack.sh
ssh -i "quant-exchange.pem" ec2-user@16.59.150.9 '
  cd ~/exchange-v2 &&
  git pull --ff-only &&
  cd exchange &&
  source "$HOME/.cargo/env" &&
  cargo build --release --bin exchange --bin market-data-service &&
  sudo cp deploy/systemd/market-data-service.service /etc/systemd/system/market-data-service.service &&
  sudo systemctl daemon-reload &&
  sudo systemctl enable market-data-service &&
  sudo systemctl restart market-data-service &&
  sudo systemctl restart exchange &&
  sudo systemctl status market-data-service --no-pager &&
  sudo systemctl status exchange --no-pager
'
```

Verify:

```bash
curl http://exchange-client-alb-1466111370.us-east-2.elb.amazonaws.com/api/health
curl http://exchange-client-alb-1466111370.us-east-2.elb.amazonaws.com/health
```

## Current backend env model

The deployed backend should define at least:

- `BIND_ADDR`
- `ADMIN_API_TOKEN`
- `CHECKPOINT_PATH`
- `CHECKPOINT_INTERVAL_SECONDS`
- `WS_BROADCAST_BUFFER`
- `WS_MARKET_BROADCAST_WORKERS`
- `MARKET_DATA_SERVICE_SOCKET=/tmp/exchange-marketdata.sock`
- `MARKET_DATA_SERVICE_RETRY_BACKOFF_MS`
- `PER_USER_REQUESTS_PER_SECOND`
