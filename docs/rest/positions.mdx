---
title: "Get Positions"
api: "GET /api/v1/positions"
description: "Read your signed net positions and realized PnL by market."
---

This is the raw positions endpoint you call when you need your current signed exposure per market.

<RequestExample>

```bash Request
curl https://exchange.jamesxu.dev/api/v1/positions \
  -H "x-api-key: YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json 200 Response
[
  {
    "market": "BTC-USD",
    "net_quantity": 40,
    "average_entry_price": 1,
    "realized_pnl": 0,
    "updated_at": "2026-03-19T21:21:34.741319Z"
  }
]
```

</ResponseExample>

## Headers

<ParamField header="x-api-key" type="string" required>
  Assigned competitor API key.
</ParamField>

## Response fields

<ResponseField name="[]" type="object[]" required>
  Current position set for the authenticated trader.
</ResponseField>

<ResponseField name="[].market" type="string" required>
  Market identifier for the position.
</ResponseField>

<ResponseField name="[].net_quantity" type="i64" required>
  Signed position size. Positive values are long and negative values are short.
</ResponseField>

<ResponseField name="[].average_entry_price" type="u64 | null" required>
  Weighted average entry price for the current open quantity. `null` when the market is flat.
</ResponseField>

<ResponseField name="[].realized_pnl" type="i64" required>
  Realized PnL from closed or settled quantity.
</ResponseField>

<ResponseField name="[].updated_at" type="string<date-time>" required>
  Last position update timestamp.
</ResponseField>
