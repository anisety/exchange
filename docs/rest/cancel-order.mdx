---
title: "Cancel Order"
api: "DELETE /api/v1/orders/{order_id}"
description: "Cancel an existing open order."
---

Call this endpoint to remove a resting order from the book.

<RequestExample>

```bash Request
curl \
  -X DELETE \
  -H "x-api-key: YOUR_API_KEY" \
  https://exchange.jamesxu.dev/api/v1/orders/ORDER_ID
```

</RequestExample>

<ResponseExample>

```json 200 Response
{
  "order": {
    "id": "66377526-7b98-485a-8c40-7024e68fa3c5",
    "trader_id": "9d64e278-8b56-46e9-9cab-18c5b22f2fb9",
    "market": "BTC-USD",
    "side": "BUY",
    "price": 100,
    "quantity": 2,
    "remaining": 1,
    "created_at": "2026-03-18T15:28:01.342510Z"
  }

```

</ResponseExample>

## Headers

<ParamField header="x-api-key" type="string" required>
  Assigned competitor API key.
</ParamField>

## Path parameters

<ParamField path="order_id" type="string" required>
  Order identifier to cancel.
</ParamField>

## Response fields

<ResponseField name="order" type="object" required>
  Final open-order state returned after the cancel succeeds.
</ResponseField>

<ResponseField name="order.id" type="string" required>
  Canceled order identifier.
</ResponseField>

## Notes

- Only open orders can be canceled.
- You can only cancel your own order.
- If the order was already fully matched or settled, the cancel will fail.
