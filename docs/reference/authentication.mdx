Your **API key** is your identity for the competition API. Organizers issue one key per competitor or team. Treat it like a password: do not commit it to public repos or share it in screenshots.

## Key format

Assigned API keys are **7-character alphanumeric strings** drawn from uppercase letters and digits (`A-Z`, `0-9`), for example `7QX2L9P`. There is no prefix. Treat the key as opaque; the length and alphabet are guaranteed by the exchange when organizers hand them out.

We recommend starting every session with [GET /api/v1/user](/rest/user) to confirm **`team_number`** and **`trader_id`**, then [GET /api/v1/portfolio](/rest/portfolio) so you know positions and limits before you send orders.

## REST (competitor routes)

Send your key on every authenticated request:

```http
x-api-key: YOUR_API_KEY
```

**These routes use the header:**

- `GET /api/v1/user`
- `GET /api/v1/portfolio`
- `GET /api/v1/balance`
- `GET /api/v1/positions`
- `GET /api/v1/open-orders`
- `GET /api/v1/fills`
- `POST /api/v1/orders`
- `PATCH /api/v1/orders/{order_id}`
- `DELETE /api/v1/orders/{order_id}`

**No key required:** [Get Markets](/rest/markets) and [Health](/rest/health).

## Rate limit (reminder)

Authenticated REST calls and authenticated WebSocket trading messages share **one per-key bucket** (default **500 ops / 10s**). See [Rate limits](/reference/rate-limits) for the exact list and error shapes.

## WebSocket

**Endpoint**

```text
wss://exchange.jamesxu.dev/ws
```

**Log in** by sending:

```json
{"op":"authenticate","api_key":"YOUR_API_KEY"}
```

**Success** looks like:

```json
{
  "type": "authenticated",
  "trader_id": "9d64e278-8b56-46e9-9cab-18c5b22f2fb9",
  "team_number": "TEAM-042"
}
```

## Good to know

- There are **no** passwords, JWTs, or OAuth flows—only your API key.
- You can connect and watch the public **`data`** stream **without** authenticating. Authenticating unlocks order entry and your private **`fill`** / **`order_state`** stream.
- Public routes such as [Get Markets](/rest/markets) do **not** consume your authenticated rate limit.
- Admin-only HTTP routes are **not** documented here; this page covers the **competitor** surface only.
